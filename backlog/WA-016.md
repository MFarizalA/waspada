# WA-016 · Rebuttal (defend_score) + ArbiterAgent ruling

## Goal

Complete the 3-round debate protocol. Round 2: the risk_model defends its score
(`defend_score()` on `RiskModelAgent`, brain `qwen3.7-plus`) — uphold or concede.
Round 3: an `ArbiterAgent` (brain `qwen3.7-max`) reads both arguments and rules
(uphold / override / escalate), or punts to the human gate on low confidence.

This turns an opened dispute (WA-014) into a fully resolved negotiation record
with the round-by-round transcript the dashboard renders.

## CUT LINE (pre-agreed)

If time runs out, **skip the Arbiter** — an upheld rebuttal escalates straight
to the human gate. The story is still complete: challenge → defense → human
rules. Document this cut in the ticket result if taken.

## Files / interfaces

- `waspada/agents/risk_model.py` — add `defend_score(dispute) -> DisputeRound`:
  - Brain `qwen3.7-plus`; reads the dispute's Round 1 challenge + the account's
    features; returns a Round 2 DisputeRound (speaker=risk_model, uphold/concede
    in the claim, confidence, cited evidence).
  - Concession → dispute resolves "overridden", resolved_by="risk_model".
  - Uphold → dispute proceeds to Arbiter (or gate, if Arbiter cut).
  - Unparsable rebuttal → auto-escalate (safe degrade).
- `waspada/agents/arbiter.py` (NEW) — `ArbiterAgent(Agent)`:
  - `name = "arbiter"`, brain `qwen3.7-max`.
  - `rule(dispute) -> (resolution, rationale, confidence)`: reads Rounds 1-2,
    returns uphold/override/escalate. Low confidence (below threshold, TBD ~0.6)
    → escalate.
  - Unparsable ruling → escalate to human.
- `waspada/agents/orchestrator.py` — wire the rebuttal + arbiter rounds into the
  dispute resolution flow; terminal resolutions map to the four states
  (upheld / overridden / escalated_approved / escalated_rejected).
- `waspada/agents/__init__.py` — export `ArbiterAgent`.

## Acceptance

- A full dispute (challenge → defend → rule) produces a 3-round `Dispute` with
  the correct terminal `resolution` and `resolved_by`.
- Concession path (Round 2 concede) → resolution="overridden", no Round 3.
- Arbiter low-confidence → escalate → human gate → escalated_approved/rejected.
- Unparsable rebuttal → auto-escalate; unparsable ruling → escalate. Both logged.
- All paths testable with a scripted MockLLM (for QA in WA-020).
- All existing tests green.
