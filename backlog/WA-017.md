# WA-017 · Efficiency benchmark (Agent Society vs single-agent baseline)

## Goal

Prove Track 3's "measurable efficiency gain over single-agent baselines" with
the honesty discipline HACKATHON.md § benchmark demands. The reworked spec is
NOT a raw speed contest — it's a decision-quality-per-cost story.

Two arms on the SAME vintage hold-out test slice (real `label_default` ground truth):
- **Single-agent baseline:** one `qwen3.7-plus` call per test account (raw
  features in → band+action out).
- **Agent Society:** sklearn scores 100% (0 LLM calls); Skeptic/Judge audit
  top-K only.

## Presentation format (what makes the claim believed)

1. **One hero number**, stated up front — target shape: *"matches the baseline's
   top-tier recall at ~⅛ the LLM calls per account."* Whatever the real run
   produces is the number — measured, never rounded up.
2. **Cost per caught high-risk account** — cost normalized by SUCCESS, not call
   volume. The honest denominator.
3. **Cost–quality frontier:** recall@call-tier (y) vs LLM-calls-per-account (x),
   sweeping K ∈ {4, 8, 16}; baseline and WASPADA as points on the same axes.
4. **Escalation rate** as a governance metric (human-review load the society
   generates — a lender cares about this as much as recall).
5. **State the honest negative:** multi-agent is NOT worth it for cheap,
   low-stakes decisions — loan-risk qualifies because each decision is
   high-value and tool-grounded. Naming the boundary is a credibility feature.

A stage that didn't run reports `not_run` — NEVER a fabricated number.

## Files / interfaces

- `waspada/bench-society/run_bench.py` (NEW) — runs both arms on the hold-out
  slice, emits the metrics.
- `waspada/bench-society/__init__.py` (NEW).
- `bench-society/AGENT_SOCIETY_BENCH.json` (committed snapshot) — the measured
  result. Present even if early: `{ "status": "not_run" }` until a real run lands.

## Acceptance

- The benchmark script runs both arms and emits: recall@call-tier,
  precision@call-tier, LLM-calls-per-account, wall-clock P50/P95, escalation rate.
- `AGENT_SOCIETY_BENCH.json` is committed with either real numbers or
  `{ "status": "not_run", "reason": "..." }`.
- The cost-quality frontier chart (K sweep) is producible from the JSON.
- No metric our cross-sectional snapshot can't produce is faked (no cure rates,
  no true band migration — HACKATHON.md honesty rails).
