# WA-052 · Model lineage — the model is retrained every run and never persisted

## The bug

`save_model()` / `load_model()` exist (`risk.py:361-373`), are exported
(`risk.py:61-62`, `model/__init__.py:14-16`) — and are called **nowhere outside
`tests/test_model.py`**. `RiskModelAgent.run()` calls `_train(frame)`
**unconditionally on every run** (`risk_model.py:65`). There is no cache, no
checkpoint, no version.

Consequences:

- **No model lineage.** You cannot reproduce *why* an account was scored as it was
  last Tuesday — the model that made that decision no longer exists.
- The audit trail meticulously records the **debate** but never records **which
  model** produced the number being debated.

For the regulated-lender pitch — *"show me the audit trail"* — an unversioned model
that silently changes between runs is a governance hole sitting **directly under**
the thing being audited.

## The fix

- Persist to `models/lane={lane}/trained_as_of=YYYY-MM-DD/model.pkl` +
  `metrics.json` — the layout **WA-047 already reserves**.
- Load-if-present; retrain only when absent or stale.
- Stamp `model_version` (artifact hash, or `trained_at`) into **both** the payload
  and the SLS audit record, so every decision names the model that made it.

## ⚠ Separate finding, surfaced during the same review — needs an owner decision

**`predict()` scores the same frame it trained on.** `_predict(model, frame)`
(`risk_model.py:77`) runs over the **full population**, including the ~70% of rows
used to fit (`risk.py:342`). So the `p_default` served to collectors is
**in-sample for most of the book**.

The hold-out AUC (`risk.py:235-237`) *is* computed correctly on the vintage split —
that number is honest. But the **served scores** are partly in-sample, which
inflates apparent separation relative to what the model would achieve on genuinely
unseen accounts.

This is a methodological issue in a *risk model*, not a bug in the plumbing, and it
deserves its own decision (out-of-fold prediction? accept and document? score only
unseen vintages?). **Flagged, not scoped here.**
