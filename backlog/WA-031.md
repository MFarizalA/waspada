# WA-031 · Dashboard EN/中文 bilingual mode + Chinese-style redesign

## Goal

Make the analyst dashboard bilingual (English / 简体中文) with a live toggle, and
re-skin it in a Chinese web-design idiom — information density, vibrant imperial
red + gold, integrated "megamenu" banner, and CJK-aware custom typography
(hierarchy via colour/weight/size/rule-lines, not the italic/bold that CJK
resists). Data values (loan ids, product/region, Q1–Q5 bands) stay untranslated.

## Files / interfaces

- `dashboard/src/lib/i18n.tsx` (NEW) — `I18nProvider` + `useI18n()` + a flat
  EN/ZH message dictionary with `{var}` interpolation. Persists to
  `localStorage`, stamps `<html lang>` + `data-lang`, auto-detects from
  `navigator.language`. `t(key, vars)` falls back to EN then the key itself.
- `dashboard/src/main.tsx` — wrap the app in `I18nProvider` (outside
  `AuthProvider` so the auth screen is localized too).
- `dashboard/src/styles/tokens.css` — retheme to imperial red (中国红) + gold +
  jade over warm porcelain; CJK sans stack + display serif; denser radii/spacing.
  **Token names kept stable** so components re-theme for free.
- `dashboard/src/App.module.css` + `App.tsx` — red-gradient megamenu banner with
  gold seam, in-page nav anchors, and the gold language toggle.
- Every string-bearing component localized via `t()`: `App`, `WorkList`,
  `PortfolioHealth`, `Alerts`, `AccountDrawer`, `AgentDialogue`, `ActionBadge`,
  `BandBadge`, `AuthScreen` (+ its `AuthScreen.module.css` red/gold restyle).
  Derived labels (severity / metric / segment / speaker / resolution) too.

## Acceptance

- A visible toggle switches the entire UI EN ⇄ 中文 live; the choice persists
  across reloads.
- No hard-coded UI chrome strings remain (data values excepted).
- `tsc -b` clean; `vite build` green.
- The redesign applies globally via tokens (cohesive, not one-off per component).

## Out of scope

- Translating backend-generated dynamic text (e.g. `Alert.message`, dispute
  rationales) — that would need backend i18n; the payload strings pass through.
- More than two locales; RTL.
